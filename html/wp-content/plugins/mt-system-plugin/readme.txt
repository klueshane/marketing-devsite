== Description ==

This is Media Temple's system plugin which provides several utility features

Key features:

* See your admin stats in your Premium WordPress Control Panel 
* Install Google Analytics UA-ID into either the site header or footer

== Installation ==

1. Sign up for Premium Wordpress Service at http://mediatemple.net and the
plugin is installed for you.
2. See your admin stats in the Media Temple Premium WordPress Control Panel!
