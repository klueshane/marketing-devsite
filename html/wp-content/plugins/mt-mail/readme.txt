== Description ==

This is a plugin for Media Temple customers to be able to access their email service from within their Wordpress Admin panel.

A new item called '(mt) Mail' is added to the side menu panel.

This allows our customers the ability to check their mail while developing their Wordpress site, without having to go thru the traditional Account Center Control Panel.

== Installation ==

comes pre-installed with your Media Temple Premium Wordpress Service
